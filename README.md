# 🚀 Minha Jornada de Aprendizado: Lógica e Programação

Bem-vindo ao meu repositório de estudos! Aqui estão concentrados os meus códigos, exercícios e projetos desenvolvidos enquanto domino os fundamentos da programação e avanço para o desenvolvimento de softwares robustos.

O repositório está organizado em duas etapas principais de aprendizado:

---

## 📂 Estrutura do Repositório

### 🇧🇷 [Portugol](./Portugol)
Esta pasta contém a minha base de tudo. Foi onde dei os primeiros passos na programação, focando puramente em **Lógica de Programação** e **Algoritmos** sem me preocupar com as complexidades de uma linguagem de mercado.
*   **Ferramenta utilizada:** Portugol Studio.
*   **O que você vai encontrar aqui:**
    *   Variáveis e operadores matemáticos.
    *   Estruturas condicionais (`se`, `senao`).
    *   Estruturas de repetição (`enquanto`, `para`).
    *   Vetores e matrizes (Estruturas de dados básicas).

### ☕ [Java](./Java)
Esta pasta representa o meu próximo nível. Aqui, transformei a lógica que aprendi em Portugol em código real para o mercado, explorando o ecossistema de uma das linguagens mais utilizadas no mundo.
*   **Ferramentas/IDEs:** VS Code / Eclipse / IntelliJ.
*   **O que você vai encontrar aqui:**
    *   Sintaxe e tipagem forte do Java.
    *   Orientação a Objetos (Classes, Objetos, Herança, Polimorfismo e Encapsulamento).
    *   Manipulação de exceções (`try-catch`).
    *   Estruturas de dados mais avançadas (`Lists`, `Maps`).
